Overview

El curso de Diseño Web de Honores es el primer curso del Programa Pathway de Diseño Web CTE. En este curso, los estudiantes aprenderán los fundamentos del Internet, principios de codificación, lenguaje de marcado de hipertexto (HTML) y hojas de estilos en cascada (CSS). Este curso preparará a los estudiantes para crear y mantener sitios web estáticos profesionales utilizando las herramientas actuales de la industria y las tendencias y tecnologías emergentes. Los estudiantes diseñarán y crearán páginas web funcionales de acuerdo con las especificaciones utilizando HTML, CSS, Bootstrap y tendencias y tecnologías emergentes. Además, los estudiantes resolverán problemas de manera colaborativa y creativa en un entorno profesional, mientras desarrollan habilidades transferibles y comercializables en preparación para carreras en el campo del diseño web.

Lo qué aprenderás

  • HTML
  • CSS
  • Bootstrap

Overview

El curso de Diseño Web Avanzado de Honores es el segundo curso del Programa de Pathway de Diseño Web de Educación Profesional y Técnica (CTE). Los estudiantes de Diseño Web aplicarán conocimientos y habilidades de codificación incorporando la aplicación del lenguaje de programación JavaScript. Los estudiantes aprenderán habilidades y estrategias de resolución de problemas para crear y mantener sitios web interactivos de acuerdo con las especificaciones del cliente. En este curso, los estudiantes obtendrán una comprensión profunda de JavaScript y otros conceptos esenciales. Estos conceptos incluyen variables, funciones, bucles, eventos y estrategias de resolución de problemas de programación.

Lo qué aprenderás

  • Programming Logic
  • JavaScript
  • Structured Learning Experience

Overview

El curso final, NJIT IS219 Advanced Website Development, presenta los conceptos y habilidades necesarios para planificar, diseñar y construir sitios web avanzados, con un enfoque en la interacción sofisticada del usuario habilitada por scripts del lado del cliente. Estos sitios web interactivos utilizan formularios para recopilar las entradas del usuario y varían tanto el contenido como la visualización de las páginas web en función de las tareas y preferencias actuales del usuario. Esto incluye diseñar y cambiar dinámicamente pestañas y menús, así como expandir y contraer secciones de páginas. Los estudiantes desarrollarán una comprensión profunda de la usabilidad del sitio web y las herramientas y habilidades que los desarrolladores web utilizan para agregar funciones interactivas a los sitios web.

Lo qué aprenderás

  • JavaScript
  • The DOM
  • JSON files
  • JQuery
  • AJAX
  • Vue.js
