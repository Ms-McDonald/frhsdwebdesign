Overview

The Honors Web Design WDPP course is the first course in the CTE Web Design Pathway Program. In this course, students will learn the fundamentals of the internet, coding principles, Hypertext Markup Language (HTML) and Cascading Styles Sheets (CSS). This course will prepare students to create and maintain professional static websites using current industry tools and emerging trends and technologies. Students will design and create functional web pages according to specifications using HTML, CSS, Bootstrap, and emerging trends and technologies. In addition, students will solve problems collaboratively and creatively in a professional environment, while developing transferable, marketable skills in preparation for careers in the field of web design.

What You’ll Learn

  • HTML
  • CSS
  • Bootstrap

Overview

The Honors Advanced Web Design WDPP course is the second course in the Career and Technical Education (CTE) Web Design Pathway Program. Web Design students will be apply coding knowledge and skills incorporating the application of JavaScript programming language. Students will learn problem solving skills and strategies to create and maintain interactive websites according to customer specifications. In this course, students will gain a deep understanding of JavaScript and other essential concepts. These concepts include variables, functions, loops, events and programming problem solving strategies.

What You’ll Learn

  • Programming Logic
  • JavaScript
  • Structured Learning Experience

Overview

The final course, Honors NJIT IS219 Advanced Website Development WDPP, introduces concepts and skills required to plan, design and build advanced websites, with a focus on sophisticated user interaction enabled by client-side scripting. These interactive websites utilize forms to gather user inputs, and vary both the content and display of the webpages based on the current user tasks and preferences. This includes designing and dynamically changing tabs and menus, as well as expanding and contracting sections of pages. Students will develop a thorough understanding of website usability and the tools and skills that web developers use to add interactive features to websites.

What You’ll Learn

  • JavaScript
  • The DOM
  • JSON files
  • JQuery
  • AJAX
  • Vue.js
